# Design archive and integration guides

The operational guides in this directory—`sidecar-modes.md`,
`terminal-adapters.md`, and `wyoming-integration.md`—describe current supported
integrations. `model-recommendations-may-2026.md` is a dated recommendation
snapshot.

Files under `plans/` and `specs/` are historical design inputs. Some shipped,
some changed during implementation, and some remain unimplemented. Do not use
them as current configuration or backlog truth without checking the runtime,
the main documentation set, and the
[July 2026 evaluation follow-up](../evaluation-follow-up-2026-07.md).
